# Data

## Important Note

The raw data (`EFA_Olderadult.xlsx`) is **not included** in this repository due to data governance agreements with BKKBN Indonesia.

## How to Access the Data

Researchers interested in accessing the PK21 2021 dataset may contact:

**Badan Kependudukan dan Keluarga Berencana Nasional (BKKBN)**
Website: https://www.bkkbn.go.id
Email: [contact BKKBN directly]

## Data File Required

Place the following file in this folder before running the analysis:

| File | Sheet | Description |
|------|-------|-------------|
| `EFA_Olderadult.xlsx` | `olderadults` | Main dataset (208,448 rows × 45 variables) |
